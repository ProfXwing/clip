# clip

A small CLI tool for copying text from the terminal.

## Usage

There are two ways to copy text.

1. As a command line argument:

    ```cli
    clip <text_to_copy>
    ```

2. By piping into clip:

    ```cli
    command_with_output | clip
    ```
